# Quick Task 260330-f91: Add Zoom to Image Display — Summary

**One-liner:** Mouse-wheel zoom (1x–8x) with QScrollArea pan/scroll on both image panels, using setFixedSize + widgetResizable toggle for scrollbar activation.

## What Was Done

Added mouse-wheel zoom and pan capability to both image display panels (original and annotated) in the PySide6 desktop app.

### Task 1: Zoom support in ScaledImageLabel (`ui/scaled_image_label.py`)

- Added `_zoom` float (default 1.0) and `_fit_size` cache attribute
- Added `zoom_in()` (×1.25, cap 8.0), `zoom_out()` (÷1.25, floor 1.0), `zoom_reset()` public methods
- Added `_apply_zoom()` which:
  - At zoom=1.0: restores flexible sizing (setMinimumSize/setMaximumSize), sets parent QScrollArea widgetResizable=True
  - At zoom>1.0: computes fit size from viewport, multiplies by _zoom, calls setFixedSize, sets parent QScrollArea widgetResizable=False
- Added `wheelEvent` that calls zoom_in/zoom_out based on angleDelta().y() and accepts the event
- Modified `clearPixmap` to call `zoom_reset()` automatically
- `paintEvent` and `mousePressEvent` unchanged — both use `self._pixmap.scaled(self.size(), ...)` which is zoom-agnostic since the label's size IS the zoomed size when fixed
- Commit: `56c1802`

### Task 2: QScrollArea wrapping and zoom reset on image switch (`ui/main_window.py`)

- Added `QScrollArea` to imports
- Wrapped `original_label` in `self.original_scroll` (QScrollArea, widgetResizable=True initially)
- Wrapped `annotated_label` in `self.annotated_scroll` (QScrollArea, widgetResizable=True initially)
- Added `zoom_reset()` calls on both labels at the top of `_on_image_selected` (before setPixmap)
- Commit: `8137070`

## Decisions Made

- **Option B (QScrollArea) chosen over Option A** (as recommended in research): scientists need to scroll to reach cells at edges of zoomed image — pure zoom without pan is useless for the task
- **No zoom buttons added**: mouse-wheel zoom is more natural; research confirmed this approach
- **Both panels zoom independently**: original and annotated have separate QScrollArea + ScaledImageLabel, each with own `_zoom` state
- **widgetResizable toggle**: at zoom=1.0, True lets label fill viewport (fit-to-widget behavior preserved); at zoom>1.0, False activates scrollbars so content larger than viewport is scrollable
