Modify ScaledImageLabel to support zoom with mouse wheel and proper sizing for QScrollArea containment.

1. Add `_zoom` float attribute (default 1.0) in `__init__`. Add `_fit_size` QSize cache attribute (None initially).

2. Add methods:
   - `zoom_in()`: multiply `_zoom` by 1.25, cap at 8.0, call `_apply_zoom()`
   - `zoom_out()`: divide `_zoom` by 1.25, floor at 1.0, call `_apply_zoom()`
   - `zoom_reset()`: set `_zoom = 1.0`, call `_apply_zoom()`
   - `_apply_zoom()`: if `_pixmap` is None return. If `_zoom == 1.0`, call `self.setMinimumSize(1, 1)` and `self.setMaximumSize(16777215, 16777215)` (QWIDGETSIZE_MAX) to restore flexible sizing. If `_zoom > 1.0`, compute the fit-scaled size as `self._pixmap.scaled(self.parent().size() if self.parent() else self.size(), Qt.KeepAspectRatio, Qt.SmoothTransformation).size()`, then set `self.setFixedSize(int(fit_w * self._zoom), int(fit_h * self._zoom))`. Then call `self.update()`.

3. Override `wheelEvent(self, event)`:
   - `delta = event.angleDelta().y()`
   - If delta > 0: `self.zoom_in()`
   - If delta < 0: `self.zoom_out()`
   - Accept the event to prevent scroll area from also scrolling.

4. Modify `paintEvent`: when `_zoom > 1.0`, scale pixmap to `self.size()` (the label's fixed size IS the zoomed size) instead of widget fit. When `_zoom == 1.0`, keep existing fit-to-widget behavior unchanged.

5. Modify `mousePressEvent`: the existing coordinate math already works because it uses `scaled.width()` / `scaled.height()` from the actual rendered pixmap. Ensure the `scaled` variable in mousePressEvent matches what paintEvent renders. When `_zoom > 1.0`, use `self._pixmap.scaled(self.size(), Qt.KeepAspectRatio, Qt.SmoothTransformation)` (same as paintEvent). When `_zoom == 1.0`, keep existing behavior.

6. Modify `clearPixmap()` to also call `zoom_reset()`.

7. Import `Qt` is already imported. No new imports needed.

Key constraint: coordinate mapping MUST remain accurate at all zoom levels. The formula `orig_x = int(img_x * self._pixmap.width() / scaled.width())` is zoom-agnostic as long as `scaled` matches what was painted.
